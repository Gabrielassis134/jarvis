# SAR

Este repositório foi criado para guardar a criação da automação no estilo jarvis pra comptador.

## Objetivo

Facilitar pequenas funções de forma rápita por meio de palmas e falas.

## Descrição

A SAR foi criada para facilitar pequenas funções, como: Play/pause de músicas e vídeos, abriri e fechar programas e aplicar atalhos como Ctrl+S e Win+L
Tudo isso sem enconstar no computador, apenas batendo palma ou falando.
Por meio de uma pequena aplicção de python a SAR detecta palmas e quando uma pessoa fala e por meio disso executa o que lhe foi programado.
